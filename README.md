## ⚙️ Arquitetura e Engenharia de Pastas

A organização deste projeto reflete o cuidado com a manutenção e escalabilidade de produtos profissionais.

* **`@api/`**: Servidor Node.js para processamento de contatos e automação SMTP.
* **`@src/hooks/`**: Lógicas de estado, hooks customizados e o motor inteligente de consentimento de privacidade.
* **`@src/pages/`**: Storytelling de cases de alta complexidade, documentando do wireframe ao código final.

---

## 🛡️ Segurança e Respeito aos Dados (GRC)

Trabalhar com dados exige responsabilidade. Este projeto foi auditado sob princípios de **Governança e Riscos**:

* **Privacidade por Design:** Implementação do **Google Consent Mode v2**. Respeito total às preferências do usuário: nenhum script de rastreio (GA4/Hotjar) é carregado sem consentimento prévio.
* **Hardening de Servidor:** Arquivo `.htaccess` configurado com **Content Security Policy (CSP)** rigorosa para bloquear ameaças digitais e garantir a integridade do site.
* **Sigilo de Negócio:** Uso de variáveis de ambiente (`.env`) para gerir segredos e links de protótipos confidenciais sob NDA.
